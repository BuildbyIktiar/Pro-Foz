# ProFoz Website

A modern and responsive business promotion website built using **HTML**, **CSS**, and **Font Awesome**. The website is designed with a clean UI, smooth sections, and professional layouts suitable for showcasing services, templates, testimonials, and creative business solutions.

## 🌐 Live Demo

Visit the website here:

`https://pro-foz.vercel.app/`

---

# 📌 Features

* Responsive modern design
* Clean and user-friendly interface
* Hero section with CTA buttons
* Service showcase section
* Creative business promotion layout
* Testimonials section
* Smooth scrolling experience
* Font Awesome icons integration
* Mobile-friendly design
* Organized HTML & CSS structure

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/profoz-website.git
```

## 2. Open the Project

Open the folder in VS Code or any code editor.

## 3. Run the Website

Simply open `index.html` in your browser.

---

# 🎨 Font Awesome Setup

Add Font Awesome CDN inside the `<head>` section:

```html
<link
  rel="stylesheet"
  href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css"
/>
```
